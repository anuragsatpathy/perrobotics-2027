# PerRobotics 2027 Website

Static workshop website for:

**Pervasive Robotics: Sensing, Intelligence, and Coordination for Human-Centered Smart Environments**

Open `index.html` locally to preview the site. The folder is ready to be used as the root of a standalone GitHub repository and published with GitHub Pages.

## Source material

- `PerCom_Workshop_Proposal.docx`
- `CFP_Workshop_PerCom2027_v4.pdf`
- Official IEEE PerCom 2027 website: <https://percom.org/>

## Publishing

1. Create a new public GitHub repository.
2. Copy the contents of this folder to the repository root.
3. Push the `main` branch.
4. In repository **Settings > Pages**, select **Deploy from a branch**, `main`, and `/ (root)`.
